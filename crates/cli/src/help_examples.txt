Comment Examples:
  Add REMIND-ME-TO comments to your source files. The tool scans for these
  and checks whether their conditions have been met.

  Basic syntax:
    // REMIND-ME-TO: <description> <operation>=<forge>:<owner>/<repo><ref>

  Any comment style works (// , # , /* */ , -- , etc.).

  Pull requests:
    // REMIND-ME-TO: remove workaround pr_merged=github:owner/repo#123
    // REMIND-ME-TO: clean up shim pr_closed=github:owner/repo#45
    // REMIND-ME-TO: drop polyfill pr_released=github:owner/repo#78

  Tags / versions:
    // REMIND-ME-TO: bump dependency tag_exists=github:owner/repo@>=2.0.0

  Commits:
    // REMIND-ME-TO: delete migration commit_released=github:owner/repo@abc1234

  Branches:
    // REMIND-ME-TO: remove feature flag branch_deleted=github:owner/repo@old-branch

  Issues:
    // REMIND-ME-TO: revisit this issue_closed=github:owner/repo#99

  Date-based:
    // REMIND-ME-TO: remove beta banner date_passed=2025-06-01

  Nixpkgs:
    // REMIND-ME-TO: switch to native package nixpkg_version=hello@>=3.0

  Multiple conditions (OR — triggers if any condition is met):
    // REMIND-ME-TO: drop compat layer pr_merged=github:owner/repo#200 tag_exists=github:owner/repo@>=3.0
